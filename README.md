# 奖学金管理系统：Backend

### 项目背景

某一年某大学的课设项目，但是当年完成的确实是不太好，于是有了此重构版本。功能更完善，更加接近企业级开发。

### 技术栈

Java, **Spring Boot**, MySQL, **MyBatis**, **Redis**, **Swagger**, **JWT**, ThreadLocal，AOP
